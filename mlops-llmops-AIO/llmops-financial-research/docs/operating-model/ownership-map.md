# Ownership Map - Financial Research Assistant

## Platform / LLMOps owns
- prompt and deployment workflow standards
- SageMaker endpoint rollout patterns
- citation and entitlement alarm integration
- rollback path for prompt, retriever, and model versions

## Research ML owns
- retrieval and rerank quality analysis
- eval dataset evolution
- prompt quality recommendations
- groundedness review support

## Product / Research Operations owns
- acceptable latency/product experience tradeoffs
- launch timing
- user feedback triage
- business value validation

## Security / Compliance owns
- entitlement expectations
- allowed corpora policy
- audit requirements for sensitive content access
