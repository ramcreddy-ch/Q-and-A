# 30 / 60 / 90 Day Onboarding Plan

## First 30 Days
- architecture review
- producer/consumer contract review
- repo setup and CI enablement
- stage config baseline

## First 60 Days
- replay validation operational
- dashboards and alarms enabled
- rollback run tested
- stage deployment repeatable

## First 90 Days
- controlled prod launch
- incident drill completed
- cost and latency review completed
- temporary exceptions reviewed and closed where possible
